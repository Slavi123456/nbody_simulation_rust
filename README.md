 ## Sandbox N-body simulation

   Самият проект представлявла интерактивен "sanbox", който с клик от мишката се създават планети.
   Телата могат да бъдат селектирани и чрез движение на мишката да им се подава инерция. При сблъсък
   в зависимост от тяхната скорост се начупват на предварително определен брой парчета като те 
   продължават да бъдат в симулацията.
